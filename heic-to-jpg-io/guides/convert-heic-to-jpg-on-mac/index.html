How to convert HEIC to JPG on Mac

macOS can export HEIC photos as JPEG without extra software. Use Preview for built-in control, Photos for library images, or the browser converter for a consistent batch workflow.

Updated July 20267 minute readBuilt-in and browser methods

Method one: export HEIC as JPEG in Preview

Preview is the most direct built-in option for a file already stored in Finder. It lets you choose JPEG explicitly, adjust output quality, and save the converted copy in a location you control.

Preview steps
  1. Open the HEIC image in Preview.
  2. Choose File > Export.
  3. Open the Format menu and select JPEG.
  4. Adjust the quality slider and choose a filename and destination.
  5. Click Save.

Apple’s Preview guide also explains that holding Option while opening the Format menu can reveal additional formats. For normal photo compatibility, ordinary JPEG is the relevant choice.

Official Apple reference

See Apple’s current instructions for converting image file types using Preview on Mac.

Batch convert several HEIC files in Preview

Preview can handle more than one image in a single window. Open the files together, select the thumbnails in the sidebar, and then use File > Export Selected Images. Choose a destination and JPEG as the output format when macOS presents the options.

Check the selection before exporting. If only one thumbnail is selected, only that file will be converted. Work on copies when the source folder is important, and choose a separate destination so HEIC originals and JPG outputs are easy to distinguish.

Method two: export from Photos

If the image is in the Photos library rather than Finder, exporting from Photos avoids an extra drag-and-drop step. Select the photo or photos, open the File menu, and use the export option that lets you choose a photo format. Select JPEG and set the available quality and size options for your goal.

Exporting creates new files outside the library; it does not need to replace the HEIC originals. That is a useful archive pattern: keep the efficient source in Photos, then create compatible JPG copies only for a client, upload, document, or recipient.

Method three: use the private browser converter

The browser tool is useful when you want the same interface across Mac, Windows, iPhone, and other devices. It also gives you per-file quality overrides and a single ZIP download for a batch.

  1. Open the HEIC to JPG converter.
  2. Drop HEIC or HEIF files onto the page.
  3. Set JPG quality and choose whether to keep EXIF metadata.
  4. Download each result, or download the batch as a ZIP.

The selected images stay on the Mac. The browser loads conversion code after you add files, decodes each image in a Web Worker, and creates the JPG in local memory. It does not send the photo to the application server.

What JPEG quality should you choose?

The right setting depends on how the image will be used. For a web form or email attachment, a setting around 78 to 84 typically keeps files practical. For general photo sharing or moderate editing, 85 to 90 gives more headroom. Very high settings produce larger files and often yield only a small visible improvement.

Compare at the actual viewing size rather than zooming far beyond 100 percent. Fine foliage, hair, text, and hard contrast edges reveal compression artifacts first. If the destination has a strict size limit, reduce quality gradually instead of making a large jump.

Will the Mac keep metadata?

Metadata handling depends on the export path and options. The browser converter offers a clear Keep EXIF control, but preservation is best effort because HEIC metadata structures vary. For sensitive public sharing, the safest choice is to disable preservation and inspect the downloaded file before posting.

For a private photo archive, dates and camera information can be valuable. Keep the original HEIC even when the JPG appears to contain the metadata you need.

Common Mac conversion problems

JPEG is missing from Preview’s Format menu

Make sure you chose Export rather than Save. If the menu appears limited, Apple recommends holding the Option key while opening the Format menu.

The output is larger than the original

This is normal at high quality. HEIC compression is generally more efficient than JPEG. Lower the JPEG quality if file size is more important than preserving every fine detail.

One image in a batch fails

Try that file separately. It may be incomplete, contain an unusual HEIF image sequence, or use a feature the selected decoder does not support. Obtain a fresh copy from Photos or the source device when possible.

A sensible Mac photo workflow

Keep HEIC masters for storage and editing, and export JPG delivery copies when a destination requires them. This avoids converting an entire library unnecessarily and preserves the most efficient source files.
