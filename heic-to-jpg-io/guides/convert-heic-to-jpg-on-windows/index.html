How to convert HEIC to JPG on Windows

The quickest method works directly in your browser and does not upload the photo. Windows also offers optional extensions when you want to view HEIC files without converting them.

Updated July 20268 minute readNo software required

Quick method: convert in your browser

If a website, document, or older program refuses an iPhone photo, creating a JPG copy is usually faster than changing the Windows setup. A browser converter is particularly useful on a shared or work computer where you cannot install extensions.

Convert now
  1. Open the HEIC to JPG converter.
  2. Drop your .heic or .heif photos onto the file area, or click it to choose files.
  3. Choose a JPEG quality. The default of 80 is a sensible balance for sharing and web uploads.
  4. Decide whether to keep EXIF metadata. Turn it off if you do not want the tool to try preserving details such as capture date or GPS.
  5. Download each JPG, or use the ZIP button after a batch finishes.

The conversion runs locally. Your selected images are read by the browser, decoded with WebAssembly, and encoded as JPEG files on the PC. The tool fetches its conversion code from public content delivery networks, but it does not send the selected photo in those requests.

Why HEIC files appear on a Windows PC

Apple devices commonly capture photos in HEIF, with .heic used as the filename extension. The format can store comparable visual quality more efficiently than JPEG, which saves device and cloud storage. Compatibility is the trade-off. A Windows application may understand the file, show only a generic icon, ask for a codec, or reject it completely.

How the image reached the PC also matters. Some sharing and transfer workflows create a compatible JPEG automatically; others preserve the original HEIC. Apple notes that importing from an iPhone or iPad can convert media in some situations, while the Keep Originals transfer setting preserves the source format. That is why two photos from the same phone can arrive with different extensions.

Option two: add HEIC viewing support to Windows

Conversion is not always necessary. If your goal is simply to open, preview, or organize HEIC files, Windows extensions may be more convenient. Microsoft’s support documentation explains that the Photos app can request HEIF and HEVC extensions when it cannot view a media type. Availability, price, and exact requirements can vary by Windows version and Store account, so follow the current prompt shown by Microsoft rather than downloading a codec from an unknown website.

Official Windows reference

Microsoft documents the Photos app extension route and alternative transfer methods in “Can’t view this file type”.

Installing viewing support leaves the original HEIC intact. That is useful for a personal library because you keep the storage-efficient source file. It does not guarantee that every third-party website or desktop program will accept HEIC. Convert a JPG copy when maximum compatibility is the actual requirement.

Choosing the right JPG quality

JPEG quality is not a percentage of the original image’s quality. It is an encoder setting that changes how aggressively the output is compressed. A very high value creates a larger file and can preserve fine texture; a lower value creates a smaller file but may show blockiness or halos around sharp edges.

  • 75–82: useful for email, forms, and general web sharing.
  • 83–90: a strong choice for photos you may edit or print at ordinary sizes.
  • Above 90: larger output with diminishing visual benefit for many everyday images.

A JPG can be larger than its HEIC source. That is not evidence that conversion failed; HEIC generally uses more efficient compression. If a form has a file-size limit, reduce quality and convert that image again.

Metadata and privacy on Windows

EXIF can contain useful information such as the camera model, exposure, orientation, date, and location. Preserve it when you rely on capture dates in a photo organizer. Remove it when you are publishing a photo and do not want to risk sharing location details. The converter’s EXIF option is best effort because HEIC containers do not all store metadata in exactly the same way.

Troubleshooting failed conversions

Confirm that the file is really HEIC or HEIF

Renaming a filename from .jpg to .heic does not change its contents. If a file was renamed incorrectly or is incomplete, a decoder may report that it contains no image. Check the original source and transfer it again.

Try a smaller batch

A compressed phone photo expands to a much larger pixel buffer while being decoded. On a low-memory PC or a browser with many tabs open, processing fewer photos at a time can prevent a tab crash.

Keep the browser online for the first conversion

The heavy decoder loads only after you choose a file. If a firewall, privacy extension, or offline connection blocks the required library download, conversion cannot start. Allow the pinned CDN resources or retry in a current browser.

Use the original photo

Edited copies, message attachments, and interrupted cloud downloads can be incomplete. If one image fails while others work, obtain a fresh copy from the phone or photo library.

Should you keep the HEIC after converting?

Usually, yes. Treat the HEIC as the efficient source and the JPG as a compatibility copy. The conversion is lossy, so repeatedly editing and resaving the JPG can gradually reduce quality. Keep the original until you know the JPG satisfies your archive, print, or delivery requirement. For a deeper comparison, read HEIC vs JPG.
